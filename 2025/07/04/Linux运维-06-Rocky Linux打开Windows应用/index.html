方法 2：使用图形界面创建虚拟机

  1. 点击”创建新虚拟机”按钮
  2. 选择”本地安装介质”
  3. 浏览并选择 Windows ISO 文件
  4. 配置内存和 CPU
  5. 选择”为虚拟机启用存储”
  6. 选择网络为”桥接网络：br0”
  7. 完成向导开始安装

结论
