常用命令速查

命令 功能
ip addr show 查看所有网络接口
ip link show 查看链路状态
ip route show 查看路由表
systemctl status systemd-networkd 检查网络服务状态
networkctl list 列出所有网络接口
networkctl status enp0s3 查看指定接口状态

注意：配置更改后必须执行 sudo netplan apply 使配置生效
