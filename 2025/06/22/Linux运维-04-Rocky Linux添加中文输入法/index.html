在 Rocky Linux 上添加并配置 ibus 中文输入法的步骤如下：

1. 安装必要组件

2. 设置系统语言环境

3. 启动 IBus 服务

4. 图形界面配置

  1. 打开 IBus 设置
    终端运行 ibus-setup。

  2. 添加中文输入法

    • 进入 Input Method 标签页
    • 点击 Add → 选择 汉语 → 智能拼音
    • 点击 Add 确认

  3. 调整输入法切换快捷键

    • 在 General 标签页设置切换快捷键（默认 Super+Space）

7. 验证输入法

  1. 打开文本编辑器（如 Gedit）
  2. 按 Ctrl+Space 或设置的快捷键切换输入法
  3. 输入拼音（如 nihao）→ 应显示中文候选词
