常用参数：

参数 作用 示例
-u <用户名> 指定 Windows 用户名 -u Administrator
-p <密码> 指定密码（明文） -p "P@ssw0rd"
-g <分辨率> 设置窗口大小 -g 1280x720
-r disk:<名>=<路径> 共享本地文件夹到远程 -r disk:linux=/home/user
-f 全屏模式（退出按 Ctrl+Alt+Enter） rdesktop -f 192.168.1.100
