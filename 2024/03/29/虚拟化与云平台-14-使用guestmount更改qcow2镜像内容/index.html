关键注意事项

  1. 权限要求：

    • 所有 guestmount 操作需要 root 权限
    • 普通用户使用时需加 sudo

  2. 分区选择：

    • 使用 virt-filesystems -a <image> 查看可用分区
    • 通常系统根分区为 /dev/sda2 或 /dev/sda3

  3. 文件系统类型：

    • 如挂载失败，可尝试指定文件系统类型：

  4. 安全卸载：

    • 修改后务必使用 guestunmount 而非常规 umount
    • 强制卸载可能导致镜像损坏

  5. 镜像验证：

    • 修改后建议启动虚拟机验证更改是否生效：
